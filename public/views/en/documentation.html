Getting started

This API is designed to perform numerical conversions between different data types. For example, converting from 10 decimal to 1010 binary or from 3.5 decimal to three point five text .

All requests must be sent to the following URL:

https://www.numberconverterapi.kodexiv.com/api/converter

Additionally, formats can be used to define the structure of the input value or the conversion result.
